Frequently Asked Questions

How do I share a file from Google Drive to embed?

Right-click the file in Drive → Share → Change to "Anyone with the link" → Copy link → paste into the Slides page and click Preview.

My preview is blank. Why?

Ensure the file is shared publicly (Anyone with the link) and that the link contains a valid file id. Some corporate accounts may also block embedding.

Can I embed PDFs, Slides and Docs?

Yes — the embed tool uses the appropriate preview URL for each file type. Slides and Docs may have specific embed URLs, but the default preview often works well.
