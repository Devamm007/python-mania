Basics

# variables
x = 5
name = "Alice"

# function
def greet(n):
		return f"Hello, {n}!"

# list comprehension
squares = [i*i for i in range(10)]
print(squares)

Common Builtins

  • len() — Get the length of an iterable.
  • sorted() — Return a new sorted list.
  • map(), filter(), zip() — Functional programming tools.
  • enumerate() — Get index and item during loops.

IO & Files

# read a file
with open('file.txt', 'r') as f:
		data = f.read()

# write to a file
with open('output.txt', 'w') as f:
		f.write('hello world')
